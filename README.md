# EcommerceApp

**A scalable e-commerce platform using Django with PostgreSQL as the primary RDBMS to support Purchasing, Tracking, Order-history, etc.**

Planned future enhancements include,

- Frontend integration using ReactJS and TailwindCSS, migration to a microservices architecture with Docker, MongoDB for flexible product data modeling and Redis for caching.

- Integration of a LLM and RAG powered chatbot to assist users with intelligent, agentic task automation. Implementing Monitoring, Logging, Observation and Evaluation services for maintenance.
